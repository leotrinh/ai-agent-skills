# Skill: branding-kit
**Phiên bản:** 1.0 | **Tác giả:** Leo
**Nguồn:** Research tổng hợp về Brand Kit, Brand Guidelines & Personal Branding

## Mô tả
Skill này hướng dẫn AI Agent tạo Brand Kit hoàn chỉnh — bộ nhận diện thương hiệu cho Landing Page/Website. Gồm **9 thành phần**, **15 checklist items**, và **7 công cụ miễn phí**. Phù hợp cho Personal Branding và doanh nghiệp vừa và nhỏ.

---

## Khi nào dùng Skill này?
- User hỏi về xây dựng thương hiệu cá nhân / doanh nghiệp
- Yêu cầu tạo Brand Guidelines / Brand Identity
- Hỏi về màu sắc, font chữ, logo cho thương hiệu
- Cần tư vấn Brand Voice, giọng văn, phong cách giao tiếp
- Yêu cầu checklist chuẩn bị Brand Kit
- Hỏi về Photos, Graphics, Icons, Charts theo brand

## Luồng tư vấn Brand Kit

```
1. NHẬN yêu cầu xây dựng brand
2. ĐỌC brand-strategy.md → Thu thập thông tin nền tảng (Mission/Vision/Values/USP)
3. XÁC ĐỊNH phase cần làm:
   Phase 1 (Nền tảng): Strategy → Brand Voice → Colors → Fonts
   Phase 2 (Visual):   Logo → Graphics & Style → Icons
   Phase 3 (Hoàn thiện): Photos → Charts → Guidelines
4. ĐỌC file reference tương ứng
5. TƯ VẤN từng bước, hỏi thêm thông tin cần thiết
6. OUTPUT: Danh sách cụ thể cần chuẩn bị / tạo ra
```

## Thứ tự thực hiện tối ưu

```
Phase 1 — Nền tảng (Làm trước)
  └── Strategy → Brand Voice → Colors → Fonts

Phase 2 — Visual (Xây dựng)
  └── Logo Design → Graphics & Style → Icons

Phase 3 — Hoàn thiện (Đóng gói)
  └── Photos → Charts → Guidelines (tổng hợp cuối cùng)
```

> ⚠️ **Lưu ý:** Guidelines làm CUỐI CÙNG vì nó tổng hợp tất cả thành phần.
> ⚠️ **Lưu ý:** Nếu chưa rõ Strategy thì các bước sau sẽ thiếu định hướng.
