# Master Checklist — Brand Kit Hoàn Chỉnh

---

## Must-have (Bắt buộc)

| # | Hạng mục | Ghi chú |
|---|---------|---------|
| 1 | ✅ Mission, Vision, Values | Nền tảng định hướng toàn bộ brand |
| 2 | ✅ Target Audience description | Cụ thể: tuổi, nghề nghiệp, nhu cầu |
| 3 | ✅ Brand Personality traits (3–5 tính từ) | VD: Authentic, Humble, Practical |
| 4 | ✅ Logo file hoặc concept direction | SVG ưu tiên; nếu chưa có thì mô tả style |
| 5 | ✅ Màu sắc yêu thích / mood keywords | VD: "tối giản, xanh biển, ấm áp" |
| 6 | ✅ Font preference / style keywords | VD: "hiện đại, dễ đọc, không cứng nhắc" |
| 7 | ✅ 3–5 bài viết mẫu Brand Voice | Để phân tích và chuẩn hóa giọng điệu |
| 8 | ✅ Ảnh cá nhân (headshot, work) | Tối thiểu 1 headshot chất lượng tốt |
| 9 | ✅ Moodboard / tham chiếu design | 3–5 thương hiệu/website/design bạn thích |

---

## Nice-to-have (Nên có)

| # | Hạng mục | Ghi chú |
|---|---------|---------|
| 10 | 💡 Tagline / Slogan | Câu ngắn gọn mô tả thương hiệu |
| 11 | 💡 Icon style preference | Outline/Filled/Duotone |
| 12 | 💡 Chart examples | Loại chart hay dùng (bar, pie, line?) |
| 13 | 💡 Competitor brands | Thương hiệu tương tự để tránh nhầm |
| 14 | 💡 Previous designs | Design cũ để tham chiếu/kế thừa |
| 15 | 💡 Social media templates đang dùng | Để maintain consistency |

---

## Thứ tự thực hiện tối ưu

```
PHASE 1 — NỀN TẢNG (Làm trước)
┌─────────────────────────────────────────┐
│  Strategy → Brand Voice → Colors → Fonts │
│  (Định hướng toàn bộ brand)             │
└─────────────────────────────────────────┘

PHASE 2 — VISUAL (Xây dựng)
┌─────────────────────────────────────────┐
│  Logo Design → Graphics & Style → Icons  │
│  (Dựa trên Strategy đã xác định)        │
└─────────────────────────────────────────┘

PHASE 3 — HOÀN THIỆN (Đóng gói)
┌─────────────────────────────────────────┐
│  Photos → Charts → Guidelines            │
│  (Guidelines làm cuối: tổng hợp tất cả) │
└─────────────────────────────────────────┘
```

> ⚠️ **Không cần làm theo số thứ tự 1–9.** Thứ tự trên (Phase 1→2→3) là tối ưu.

## Câu hỏi nhanh để AI Agent thu thập thông tin

Khi user yêu cầu làm Brand Kit, hỏi theo thứ tự:

```
1. "Thương hiệu của bạn tên gì và hoạt động trong lĩnh vực nào?"
2. "Đối tượng bạn muốn phục vụ là ai? (tuổi, nghề, nhu cầu)"
3. "Tại sao họ nên chọn bạn thay vì người khác? (USP)"
4. "Nếu dùng 3-5 tính từ để mô tả thương hiệu, bạn chọn gì?"
5. "Bạn thích phong cách visual của brand nào? (3-5 ví dụ)"
6. "Bạn đã có logo chưa? Nếu chưa, style nào bạn thích?"
7. "Bạn chủ yếu sẽ dùng brand này trên kênh nào?"
```

## Điểm kiểm tra cuối cùng

Trước khi coi Brand Kit là hoàn chỉnh, kiểm tra:

- [ ] Logo hiển thị đúng trên nền sáng VÀ tối
- [ ] Màu sắc pass Contrast Ratio 4.5:1
- [ ] Font hỗ trợ đầy đủ dấu Tiếng Việt
- [ ] Brand Voice nhất quán qua 5 bài viết mẫu
- [ ] Có đủ: headshot, work photo, lifestyle photo
- [ ] Templates Social Media đúng kích thước chuẩn
- [ ] Guidelines đã document tất cả do's & don'ts
- [ ] Test trên mobile (font size, contrast, layout)
